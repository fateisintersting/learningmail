### Core Concept: HashMap (Dictionary)
A HashMap stores key-value pairs and retrieves data in constant time. 
It uses a hashing function to map a key to an index. Think of it like 
a hotel room system—your room number takes you straight to the door. 
It’s used in caching, compilers, routing tables, and more.

### DSA Question: Two Sum  
Input: [2, 7, 11, 15], target: 9  
Output: [0, 1]

Explanation:  
Use a HashMap to store each number and its index. For every number x, 
check if target - x exists in the map.

```js
function twoSum(nums, target) {
  const map = new Map();
  for (let i = 0; i < nums.length; i++) {
    const diff = target - nums[i];
    if (map.has(diff)) return [map.get(diff), i];
    map.set(nums[i], i);
  }
}
